Create docker network named gdi

Create user by docker exec into gdi_gdi.web_1 and run meteor in /app
Then start another command prompt and also docker exec into before mentioned container.
Start a meteor shell with meteor --allow-superuser shell and add a user with: Accounts.createUser({username: [username], password: [password]});
Finally add a logo to the container with: docker cp [username].png gdi_gdi.web_1:/var/lib/geodropinlogos

tnsnames.ora in ogr2ogr container
provider.conf in provider container

Files to be manually added:

trusted.jks & private.jks in service container

Create a separate conf file for each customer in /docker/nginx/conf
